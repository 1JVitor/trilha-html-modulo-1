Diferença entre Aplicação e Aplicação Web

  • Aplicações requerem o download de algum aplicativo no dispositivo para que possa ser acessada
  • Aplicações Web não necessitam desse download, e podem ser acessadas através de navegadores que possuem a internet como meio de comunicação entre cliente e server.

Sobre os Servers

O conteúdo apresentado pela DIO foi bem didático, mostrando as possibilidades dos servers, podendo como exemplo obter essas estruturas:

  • Clients - Proxy - Internet - Server
  • Clients - Firewall - Internet - Server
  • Clients - Internet - Server
  • Clients - Server em uma estrutura Intranet

Os Servers podem ser classificados e no projeto pode ser sim que se use mais de um para cumprir o objetivo da aplicação web.

Linguagens Client Side e Server Side

A diferença entre os dois termos se da pelas linguagens utilizadas para comunicação com cada "lado".

  • Client Side temos linguagens como JavaScript, CSS, HTML para comunicação com o Browsers.
  • Em Server Side temos as linguagens que se comunicam com os servidores sendo elas elas: PHP, Ruby, JavaScript, C#.

O JavaScript pode ser executado de um lado dos dois lados.
